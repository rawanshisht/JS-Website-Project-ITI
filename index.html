A Document Type Declaration, or DOCTYPE for short, is an instruction to the web browser about the version of markup language in which a web page is written. The doctype declaration itself is not an HTML tag. In HTML 4.01, the declaration refers to a DTD, because HTML 4.01 was based on SGML. The DTD specifies the rules for the markup language, so that the browsers render the content correctly. HTML5 is required.: Always add the declaration to your HTML documents, so that the browser knows what type of document to expect.
